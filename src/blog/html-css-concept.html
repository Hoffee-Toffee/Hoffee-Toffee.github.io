CSS Positioning

Relative, Absolute, and Fixed positioning

Absolute

Absolute positioning means to place the element relative to the first parent up the chain that has position relative or absolute (or the page itself if none).
This is very useful, it is like choosing coordinates within these bounds as to place the element.
Think of it as choosing coordinates within these bounds to place the element.

For example, if you do... ...then it will place the element 5em from the top of the bounds (nearest parent that fits the conditions above), which is the page in this context. Absolute: (Relative to page)

This will also remove it from the 'flow' meaning that it won't push other elements where it was.

Fixed

Fixed positioning means to place the element relative to screen, unaffected by scroll.
Think of it like it's coordinates on your physical screen, as if it were a sticky note placed on the screen itself.

For example, if you do... ...then it will place the element on the screen, halfway from the bottom. Fixed: (Relative to screen)
Very handy for navigation bars and footers.
Also taken out of flow.

Relative

Relative positioning means to place the element relative to itself.
Think of it as simply offsetting.

For example, if you do... ...then it will shift the element down by this amount.
This is compared to it's original position.
